# CatOnWindow
Кот для твоего рабочего стола

__Описание__
Это питомец для твоего рабочего стола. Он очень похож на тамогочи (честно говоря это и есть попытка повторить его).

__Что можно с ним сделать?__
ГЛАДИТЬ - при наведении на котика и прокрутке колёсика вверх -> погладить против шерсти,
          при прокрутке колеса вниз -> погладить по шерсти.
ПЕРЕНОСИТЬ - просто зажми правую кнопку мыши на котике и перенеси его на другое место.
              Будь осторожен котика можно укачать :D
